Using custom fonts in Tailwind CSS

Let's look at how to use custom font with Tailwind CSS! I'll show you how to use a Google webfont, aswell as self-hosted fonts, using the @font-face CSS at-rule. By the end of this screencast, you'll feel comfortable using any custom font for your next project.

Customizing font families

By default, Tailwind provides three fonts utililties: a cross-browser sans-serif stack, a cross-browser serif stack, and a cross-browser monospace stack. You can change add, or remove these by editing the theme.fontfamily section of your Tailwind config.
